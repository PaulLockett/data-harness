# case_001 — Polars DataFrame baseline

A 5-row synthetic Polars DataFrame with three columns (`x`, `y`, `label`).
Exercises the `kind = "DataFrame"` branch of `helpers.glance()` and validates:

- shape is plausible (3-100K rows, 2-200 cols)
- schema key set includes the 3 columns we know we put in
- null counts are returned per column
- summary string starts with `pl.DataFrame` (regex predicate; refutes a bug
  that would return a generic `peek` string instead of the deep `glance` path)
- top-level dict has all 7 expected keys

Generated by `gen_fixture.py` in this folder. Re-run if the fixture is lost.
